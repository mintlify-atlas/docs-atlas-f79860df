# Welcome to Quail BI

Quail is an AI-powered data analysis and visualization platform that enables you to query SQL databases, generate charts, and gain business insights using conversational AI. Built with Next.js and TypeScript, Quail makes data exploration accessible through natural language.

## Why Quail?

Quail bridges the gap between complex data and actionable insights by combining:

- **Natural Language Processing**: Query your data using plain English instead of SQL
- **AI-Powered Insights**: Get automated explanations and recommendations
- **Secure Architecture**: Read-only queries with encrypted credential storage
- **Modern Stack**: Built with Next.js 15, React 19, and Azure AI
